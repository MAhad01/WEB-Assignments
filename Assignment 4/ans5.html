HTML5 Video

HTML5 introduced the 'video' element for embedding videos. It is an easier and more flexible way of embedding media on webpages.



HTML5 Canvas

The 'canvas' element is used to draw graphics on a web page. It is only a container for graphics, JavaScript is used to draw.

Your browser does not support the HTML canvas tag.


HTML5 Geolocation

The Geolocation API allows the user to provide their location to web applications if they so desire.
