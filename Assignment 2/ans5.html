MERN Stack (MongoDB, Express.js, React.js, Node.js)
A JavaScript tech stack that allows developers to build fully functional web apps using the JavaScript language and JSON.
MEAN Stack (MongoDB, Express.js, Angular, Node.js)
Similar to MERN but uses Angular for the front-end JavaScript framewor.
LAMP Stack (Linux, Apache, MySQL, PHP)
A popular open-source web development platform that uses Linux as the operating system, Apache as the web server, MySQL as the relational database management system and PHP as the object-oriented scripting language
MEVN Stack (MongoDB, Express.js, Vue.JS, Node.js)
Similar to MERN and MEAN but uses Vue.JS for the front-end JavaScript framework
Ruby On Rails
A server-side web application framework written in Ruby. It is a model-view-controller (MVC) framework that provides default structures for a database, a web service, and web pages
Django
A high-level Python web framework that enables rapid development of secure and maintainable websites. Built by experienced developers, Django takes care of much of the hassle of web development
.NET
A free, cross-platform, open-source developer platform for building many different types of applications. With .NET, you can use multiple languages, editors, and libraries to build for web, mobile, desktop, games, and IoT
JAMSTACK
A modern architecture for building websites; JavaScript handles any dynamic programming during the request/response cycle, reusable APIs are accessed over HTTPS with JavaScript, and Markup is prebuilt at deploy time
